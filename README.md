# Reservation App Example

This repository contains a minimal example of a Flutter reservation system. It demonstrates:

- A simple registration screen where a user can enter their name and pick a date/time.
- Storage of reservation data locally using the [`sqflite`](https://pub.dev/packages/sqflite) package.
- Listing of existing reservations stored in the database.

## Getting Started

1. Ensure you have Flutter installed. See [Flutter installation guide](https://docs.flutter.dev/get-started/install) for details.
2. Install dependencies:
   ```bash
   flutter pub get
   ```
3. Run the application:
   ```bash
   flutter run
   ```

This will launch the app on your connected device or emulator.
